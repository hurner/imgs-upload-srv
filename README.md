> #### 在线图片服务

- Why

    ```
    1. 使用markdown时，需要引入在线的图片
    2. 发送邮件或者相关网页时需要引入在线图片
    3. 需要截图保存相关图片，并在日后随时查看下载
    ...
    ```

- #### [imgs.thankjava.com](https://imgs.thankjava.com)
```
    免费的，简洁的图片在线服务。
即开即用，开源免费。基于GNU通用公共许可证v3.0，提供在线图片服务解决方案。
```
